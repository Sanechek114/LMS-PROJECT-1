# **Приложение для учёта сельскохозяйственных животных.**

## ТЗ 
Сделать приложение для учета сельскохозяйственных животных.
## О проекте
- Приложение позволяет удобно просматривать, вести учёт данных о сельскохозяйственных животных:
  *записывать имя животного, номер, пол, вид, дату рождения и другое.*
- Интерефейс сделан на PyQt6.
- Таблицы хранятся с помощью базы данных SQLite
## Установка зависимостей
```bash
pip install PyQt6
```
# Пояснительная записка.
## Главное меню.
- Строка ввода дя поиска по таблице.
- Кнопки Добавить и Удалить для изменения строк в таблице.
- Таблица.
- Виджет для добавления строк в таблицу.
  *Открывется на кнопку добавить.*
